Quick Integration Guide & Example Code

Link the offline files locally inside your HTML projects. Below is a complete starter template showing the required loading sequence:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Responsive Website</title>
  
  <!-- 1. Include Grid + Spacing Stylesheet -->
  <link rel="stylesheet" href="ak-bootstrap.min.css">
</head>
<body class="ak-bg-light">

  <div class="ak-container ak-py-40">
    <h1 class="ak-h1 ak-text-primary">Hello World!</h1>
    <p class="ak-text-secondary">This page is styled using offline ak.css Spacing engine.</p>
    
    <!-- Example vector icon usage -->
    <ak-icon name="lucide/zap" size="24px" color="#00D4FF"></ak-icon>
  </div>

  <!-- 2. Include JIT Compiler Script (Must load before icon pack) -->
  <script src="ak-bootstrap.min.js"></script>

  <!-- 3. Include your desired icon pack (e.g. Lucide Icons) -->
  <script src="ak-icons-lucide.min.js"></script>
</body>
</html>
